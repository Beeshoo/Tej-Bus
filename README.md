# نظام تيج باص - Taj Bus Management System 🚌

نظام متكامل لإدارة وحجز حافلات النقل السياحي والرحلات بين المحافظات المصرية، مصمم لتقديم تجربة مستخدم فاخرة وسهلة.

## 🚀 نظرة عامة على المشروع
تيج باص هو نظام برمجي مبني باستخدام تقنيات الويب الحديثة، يهدف إلى رقمنة عملية حجز التذاكر وإدارة أسطول الحافلات. يوفر النظام واجهات مخصصة لكل من (المسافرين، السائقين، والمديرين) لضمان تدفق عمل سلس وفعال.

## ✨ المميزات الرئيسية

### 👤 واجهة المستخدم (User Side)
- **نظام حجز تفاعلي:** اختيار المنطلق والمقصد مع حساب آلي للسعر والمسافة.
- **اختيار المقاعد:** واجهة رسومية لاختيار المقاعد المفضلة (مع منع الحجز المزدوج).
- **سجل الرحلات:** عرض التذاكر السابقة والقادمة مع إمكانية طباعة التذكرة (QR Code).
- **نظام التقييم:** إمكانية تقييم الرحلة والسائق بعد الوصول.
- **مركز الدعم:** دردشة مباشرة ونظام لتقديم الشكاوى والمقترحات.

### 👮 واجهة السائق (Driver Portal)
- **جدول الرحلات:** عرض الرحلات المسندة للسائق من قبل الإدارة.
- **تفاصيل الركاب:** رؤية قائمة الركاب والمقاعد المحجوزة لكل رحلة.

### 🛠️ لوحة التحكم الإدارية (Admin Dashboard)
- **الإحصائيات:** نظرة عامة على الإيرادات، عدد الرحلات، والمستخدمين النشطين.
- **إدارة الرحلات:** التحكم في الحجوزات وإسناد السائقين للحافلات.
- **إدارة السائقين:** إضافة أو حذف السائقين ومتابعة حالتهم.
- **إدارة الشكاوى:** نظام للرد على شكاوى المستخدمين وإغلاقها.
- **الدردشة المباشرة:** التواصل الفوري مع المستخدمين لحل مشكلاتهم.

## 🛠️ التقنيات المستخدمة (Tech Stack)

- **Frontend:** React 19 (Functional Components & Hooks)
- **Build Tool:** Vite (لسرعة البناء والتطوير)
- **Language:** TypeScript (لضمان جودة الكود وتجنب الأخطاء المنطقية)
- **Styling:** Tailwind CSS (تصميم متجاوب وعصري)
- **Animations:** Framer Motion (لإضافة سلاسة في الانتقالات)
- **Database:** Dexie.js / IndexedDB (تخزين البيانات محلياً في المتصفح لضمان السرعة والعمل Offline)
- **Utilities:** Lucide React (أيقونات), HTML5-QRCode (لقراءة التذاكر)

## 📁 هيكل المشروع (Project Structure)
- `src/components/`: يحتوي على مكونات الواجهة (Navbar, Dashboard, BookingFlow, etc.)
- `src/services/`: يحتوي على منطق التعامل مع قاعدة البيانات (api.ts, db.ts)
- `src/types.ts`: تعريف الواجهات النوعية (Interfaces) لضمان تناسق البيانات.
- `src/constants.tsx`: الثوابت ومنطق الحسابات (الأسعار، المناطق، المواعيد).
